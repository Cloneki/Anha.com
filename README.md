# Dr. Anha - Cinematic Tribute Website

Single-page cinematic website built with HTML, Tailwind CSS, GSAP, and Three.js.

## Files

- `face.html` - main website file

## Run Locally

1. Clone/download this repository.
2. Open `face.html` in a browser.

## Features

- Cinematic scroll-based sections
- 3D animated scenes (bangles, dog, badminton, about/neural theme)
- Responsive layout for mobile, tablet, and laptop

## Deploy

You can deploy this directly as a static site (GitHub Pages, Netlify, Vercel static hosting, etc.) using `face.html` as the entry page.
